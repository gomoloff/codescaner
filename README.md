# CodeAnalyzer_v2

Многоплатформенный анализатор исходного кода для консоли, десктопа и мобильных устройств.

## Описание

CodeAnalyzer_v2 — это инструмент для анализа и статистики исходного кода. Приложение сканирует указанные папки, собирает информацию о файлах и генерирует отчёты в различных форматах.

## Компоненты проекта

Проект состоит из нескольких модулей:

### 📦 CodeAnalyzer.Core
Базовая библиотека с основной логикой анализа. Содержит:
- Конфигурацию анализатора
- Статистику анализа
- Модели данных

### 💻 CodeAnalyzer.Cli
Консольное приложение для работы через командную строку. Поддерживает:
- Интерактивный режим настройки
- CLI-режим с аргументами командной строки
- Гибкую настройку параметров анализа

### 🖥️ CodeAnalyzer.WinForms
Приложение с графическим интерфейсом на базе Windows Forms.

### 📱 MauiAppCross_v2
Кроссплатформенное мобильное приложение на .NET MAUI для анализа кода на устройствах Android, iOS и других платформах.

## Требования

- .NET 8.0 или выше
- Для MAUI-приложения: соответствующая платформа (Android, iOS, Windows, macOS)

## Сборка и запуск

### Консольная версия
```bash
cd CodeAnalyzer.Cli
dotnet run
```

### WinForms версия
```bash
cd CodeAnalyzer.WinForms
dotnet run
```

### MAUI версия
```bash
cd MauiAppCross_v2
dotnet build
```

## Использование

### Интерактивный режим
Запустите приложение без аргументов для пошаговой настройки:
1. Укажите папки для анализа
2. Настройте исключения (bin, obj, node_modules и т.д.)
3. Выберите выходную папку для отчёта
4. Настройте максимальный размер части отчёта
5. Выберите режим расширений файлов

### CLI режим
Передайте параметры через командную строку для автоматизированного запуска.
